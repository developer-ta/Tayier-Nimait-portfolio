# Plan d'action du jour

_Objectif : travailler en priorité sur le profil et surtout sur le portfolio, avec un ordre clair et des livrables concrets._

## 1. Ce que tu dois retenir aujourd'hui

Le document `Strategie LinkedIn, ATS et Positionnement 2026` est une base solide.

Mais pour avancer vraiment, il faut faire **2 choses aujourd'hui** :

1. mettre ton profil au bon niveau,
2. préparer la page projets de ton portfolio.

Le but n'est pas de tout finir.

Le but est de rendre le portfolio lisible, crédible et actionnable.

## 2. Priorité du jour

### Priorité 1

Travailler le **profil public** :

- titre LinkedIn,
- section À propos,
- compétences visibles,
- phrase de positionnement.

### Priorité 2

Travailler le **portfolio principal** :

- page projets,
- cartes projet,
- texte de chaque projet,
- ordre d'affichage.

### Priorité 3

Préparer la **preuve** :

- démo,
- capture,
- lien,
- extrait de résultat.

## 3. Ce que tu dois faire dans l'ordre

### Étape 1 - Ouvrir les bons fichiers

Commence par ces trois documents :

- `Strategie LinkedIn, ATS et Positionnement 2026`
- `Plan CV et Portfolio 2026`
- `Roadmap Source de Verite 2026`

Ces trois fichiers servent de base commune.

### Étape 2 - Stabiliser ton positionnement

Tu dois garder une seule ligne claire :

> Développeur IA appliquée orienté Python, automatisation et outils métier.

Tu dois ensuite ajouter la stack visible :

- Python
- FastAPI
- React
- API REST

### Étape 3 - Réécrire le profil LinkedIn

Tu dois mettre à jour 4 zones :

#### Titre

Format conseillé :

> Développeur IA appliquée | Python, FastAPI, React | Automatisation & outils métier

#### À propos

Tu dois dire :

- ce que tu fais,
- avec quelle stack,
- pour quel problème,
- quel type d'opportunité tu cherches.

#### Compétences

Tu gardes une liste courte et ciblée :

- Python
- FastAPI
- React
- API REST
- Automatisation
- IA appliquée
- LLM
- Outils métier

#### Projets en vedette

Tu gardes seulement 3 projets maximum.

## 4. Ce que tu dois faire pour le portfolio

Le portfolio est le vrai chantier du jour.

### Objectif

Transformer les projets en preuves simples.

### Structure d'une page projet

Chaque projet doit suivre exactement ce format :

1. **Nom du projet**
2. **Problème**
3. **Solution**
4. **Stack**
5. **Résultat**
6. **Preuve**

### Exemple de phrase courte

> J'ai automatisé un processus manuel pour réduire le temps de traitement et rendre le résultat exploitable plus vite.

## 5. Les 3 projets à préparer aujourd'hui

Tu n'as pas besoin d'en montrer 10.

Tu dois préparer 3 projets piliers :

### Projet 1

Automatisation de données ou de process.

### Projet 2

Dashboard ou outil de pilotage.

### Projet 3

Assistant IA, RAG, workflow ou outil métier intelligent.

## 6. Ce qu'il faut écrire pour chaque projet

Pour chaque projet, tu dois écrire :

- le besoin réel,
- ce que tu as construit,
- comment tu l'as construit,
- ce que ça apporte,
- ce qu'on peut voir ou tester.

### Formule simple

> Problème -> Solution -> Résultat -> Preuve

## 7. Ce que tu dois éviter aujourd'hui

- écrire des descriptions trop longues,
- ajouter trop de jargon,
- empiler trop de technologies,
- créer trop de nouveaux documents,
- mélanger profil, stratégie et archive.

## 8. Ce que tu dois produire aujourd'hui

### Livrables minimum

1. un titre LinkedIn propre,
2. un texte À propos propre,
3. 3 projets préparés pour la page projets,
4. 1 ordre clair pour afficher les projets,
5. 1 version courte de ton positionnement.

### Livrables idéaux

1. profil LinkedIn mis à jour,
2. page projets structurée,
3. première version de démo ou capture,
4. page portfolio cohérente.

## 9. Ordre de travail recommandé

### Bloc 1 - 30 minutes

- relire le positionnement,
- choisir le titre,
- fixer la phrase centrale.

### Bloc 2 - 45 minutes

- réécrire LinkedIn,
- remplir l'À propos,
- choisir les compétences.

### Bloc 3 - 60 à 90 minutes

- préparer les 3 projets,
- écrire problème / solution / résultat,
- choisir les preuves.

### Bloc 4 - 30 minutes

- vérifier la cohérence globale,
- retirer ce qui est trop large,
- garder seulement ce qui vend.

## 10. Le bon critère de réussite

Ton travail du jour est réussi si quelqu'un peut comprendre en moins de 10 secondes :

- qui tu es,
- ce que tu fais,
- ce que tu sais construire,
- pourquoi ton profil est utile.

## 11. Résumé ultra court

Aujourd'hui :

1. tu fixes ton positionnement,
2. tu mets à jour LinkedIn,
3. tu prépares la page projets,
4. tu gardes seulement 3 projets piliers,
5. tu transformes tes projets en preuves.
