# Assets projets

Arborescence simple pour les images et GIF des projets du portfolio.

## Convention

- `cover.png` : image principale de la carte projet
- `preview.gif` : courte animation de démonstration
- `thumb.webp` : miniature légère si besoin

## Projets prioritaires

- `smart-facturation-ai/`
- `dialogue-digest-agent/`
- `solena/`
- `python-mastery-nexus/`
- `tayier-core-kernel/`
- `agent-diagnostic-prompt-injection/`

## Règle

- une image claire par projet
- un GIF court par projet
- pas de capture de code brut
