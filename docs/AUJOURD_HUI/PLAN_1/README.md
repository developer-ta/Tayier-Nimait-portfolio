# Plan 1 - Inventaire des projets

_Objectif : faire la premiere selection des projets 2025-2026 a mettre dans la vitrine publique du portfolio._

## 1. But

Trouver les projets qui doivent etre visibles publiquement.

## 2. Ce que tu fais

1. Parcourir la racine de tes projets 2025-2026.
2. Relever tous les projets qui ont une vraie valeur vitrine.
3. Marquer les projets sensibles ou confidentiels.
4. Classer chaque projet dans une seule categorie :
   - `Vitrine`
   - `Secondaire`
   - `Archive`
   - `Prive`

## 3. Ce qu'on cherche

Pour chaque projet, verifier s'il montre :

- du fullstack,
- de l'IA appliquee,
- de l'automatisation,
- une solution concrète,
- une preuve visuelle,
- une valeur scientifique, medicale, biomecanique ou robotique si pertinente.

## 4. Regle de selection

Un projet va en `Vitrine` seulement si :

- il est facile a comprendre,
- il montre une competence visible,
- il est utile pour un recruteur,
- il peut etre presente sans divulguer de secret.

## 5. Format de note pour chaque projet

```text
Nom :
Categorie :
Pourquoi :
Sensibilite :
Preuve visuelle :
```

## 6. Sortie attendue

A la fin de ce plan, tu dois avoir :

- une liste de projets vitrine,
- une liste de projets secondaires,
- une liste de projets archives,
- une liste de projets prives.
